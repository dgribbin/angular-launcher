Getting Started New User

If this is the first time using the AVEVA Cloud UE service, in order to securely access your dedicated AVEVA Cloud UE WorkSpace, please

looks_one Visit the Amazon Web Services WorkSpaces site and download and install the latest Amazon WorkSpace client application for Windows from the AWS web site. This enables AVEVA to delivery secure, high performance Cloud UE services.

  • looks_two Download and Install the AVEVA Cloud UE integration client.

  • You are now ready to created your dedicated Cloud UE desktop. Click the Create button below to start this proces. This may take up to 20 minutes, so please be patient.
